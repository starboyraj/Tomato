# 🍔 Tomato – Full Stack Food Delivery Web Application

Tomato is a complete Full Stack Food Delivery Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) with Stripe Payment Integration.  
The platform allows users to browse food items, add them to the cart, place orders with secure online payments, and track order status. It also includes an Admin Panel for managing food items and orders.

---

## 🚀 Features

### User Features
- User Authentication (Signup / Login)
- Browse food items by categories
- Add / Remove items from cart
- Dynamic Shopping Cart
- Secure online payment using Stripe
- Order placement system
- Order history tracking

### Admin Panel
- Add new food items
- Upload food images
- Manage food menu
- View all orders
- Update order status (Food Processing, Out for Delivery, Delivered)

### Payment
- Secure Stripe Payment Gateway integration
- Online payment processing
- Order confirmation after successful payment

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/starboyraj/Tomato-FOOD-DELIVERY.git

2. Install dependencies

Frontend
cd frontend
npm install

Backend
cd backend
npm install

Admin Panel
cd admin
npm install

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder and add:

MONGODB_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  
STRIPE_SECRET_KEY=your_stripe_secret_key  

---

## ▶️ Run the Project

Backend
npm run server

Frontend
npm run dev

Admin Panel
npm run dev
