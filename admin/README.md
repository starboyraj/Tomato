# Hungry – Food Delivery App (Frontend)

A modern and responsive food delivery web application inspired by Swiggy, built using React.js.

## 🚀 Live Demo
https://hungries-food-delivery.vercel.app/

## 📌 Features
- Responsive UI for mobile and desktop
- Food listing and category sections
- Shopping cart UI with quantity control
- Clean and reusable React components
- Modern UI/UX design

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

## 📂 Project Structure
src/
├── components/
├── pages/
├── assets/
├── context/
├── App.jsx
└── main.jsx

## 🧠 What I Learned
- Component-based architecture in React
- State management using hooks
- Reusable UI components
- Building scalable frontend structure
